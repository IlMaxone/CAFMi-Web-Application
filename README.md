<h1 align="center">Final Project Work - CAFMi</h1>
<img src="files/img/CAFMi-logoBanner.png" alt="logo e banner Cafmi">
 
<p>
    Benvenuto nella sezione GitHub del progetto CAFMi! <br>
    Come ricorda anche il nome del progetto, la Web App nasce per risolvere delle reali problematiche che riscontrano le istituzioni
    come il CAF ed i suoi clienti riguardo la gestione della burocrazia italiana.
</p>
 
<p>
    Esistono infatti delle pratiche che richiedono delle lunghe tempistiche prima di essere processate e, per risolvere questa problematica,
    insieme a tutto il team abbiamo realizzato questa applicazione che senz'altro aiuterà a migliorare i tempi delle pratiche e,
    aggiungendo anche le funzionalità per la gestione interna dell'azienda, permetterà ai CAF di avere un workflow SMART e VELOCE.
</p>
 
<hr>
